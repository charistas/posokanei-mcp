# Security Policy

## Supported Versions

Security fixes target the latest published version.

## Reporting a Vulnerability

Open a private security advisory on GitHub:

https://github.com/charistas/posokanei-mcp/security/advisories/new

## Scope

This MCP server is intended to be read-only. Reports about accidental writes, credential leakage, prompt-injection-prone tool descriptions, or unbounded API calls are in scope.

Do not use this project to overload or abuse PosoKanei infrastructure.
